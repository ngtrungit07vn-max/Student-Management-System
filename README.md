# Student-Management-System

## Description
Hệ thống quản lý sinh viên được xây dựng bằng ngôn ngữ C++, hỗ trợ quản lý thông tin sinh viên thông qua các chức năng cơ bản như thêm, sửa, xóa và tìm kiếm dữ liệu.
## Features
- Add student
- Edit student
- Delete student
- Search student
- Save and load data from file

## Technologies
- C++
- File handling
- Data structures (struct, vector)
